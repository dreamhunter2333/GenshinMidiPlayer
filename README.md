# 原神 MiDi 弹琴

## 使用

keymap.json 为琴键映射

```bash
python3 -m venv ./venv
./venv/bin/python3 -m pip install -r requirements.txt
./venv/bin/python3 main.py -f 打上花火.mid -b 120 -w 5 --keymap_file keymap.json
```
